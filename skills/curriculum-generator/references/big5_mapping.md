# Big5 → 学習スタイル マッピング

> Rise Path パーソナライゼーションエンジンの中核ルール。
> カリキュラム生成時に学習者のBig5プロフィールに基づいてコンテンツを最適化する。

## マッピングルール

### 開放性 (Openness)

| スコア | 学習スタイル | 具体的な調整 |
|:---|:---|:---|
| 高 (70-100) | 探索型・抽象型 | メタファー多用、異分野とのクロスオーバー、「なぜ」を重視 |
| 中 (40-69) | バランス型 | 理論と実践のバランス、適度な応用問題 |
| 低 (0-39) | 実践型・具体型 | 明確な手順書、具体例先行、変化を最小限に |

### 誠実性 (Conscientiousness)

| スコア | 学習スタイル | 具体的な調整 |
|:---|:---|:---|
| 高 (70-100) | 構造型・計画型 | チェックリスト、明確なゴール、進捗バー、ステップバイステップ |
| 中 (40-69) | 柔軟型 | ガイドライン提供しつつ自由度も確保 |
| 低 (0-39) | 体験先行型 | **成功体験 → 理論の順序**、短いスプリント、即座のフィードバック |

### 外向性 (Extraversion)

| スコア | 学習スタイル | 具体的な調整 |
|:---|:---|:---|
| 高 (70-100) | 対話型・協調型 | ディスカッション課題、ペア作業、エネルギッシュなトーン |
| 中 (40-69) | 適応型 | 個人作業と協調作業のミックス |
| 低 (0-39) | 内省型・独立型 | 個人ワーク重視、振り返りジャーナル、静かなトーン |

### 協調性 (Agreeableness)

| スコア | 学習スタイル | 具体的な調整 |
|:---|:---|:---|
| 高 (70-100) | 支援型 | 励ましのメッセージ、チームワーク課題、コミュニティ感 |
| 中 (40-69) | バランス型 | 適度な競争と協調のミックス |
| 低 (0-39) | 独立型・批判型 | 論理的な根拠重視、独力での問題解決、チャレンジ課題 |

### 神経症傾向 (Neuroticism)

| スコア | 学習スタイル | 具体的な調整 |
|:---|:---|:---|
| 高 (70-100) | 安心型 | 失敗を恐れない環境作り、段階的な難易度上昇、警告と対策の提示 |
| 中 (40-69) | 標準型 | 適度なチャレンジと安全網のバランス |
| 低 (0-39) | チャレンジ型 | 高難度タスク、プレッシャー下での実践、自己管理 |

## 複合パターン (よくある組み合わせ)

### 「せっかちなクリエイター」 (開放性↑ × 誠実性↓)
→ 体験先行 + 抽象的探索。**まず手を動かして面白い結果を見せ、その後で理論を説明する。**

### 「慎重な学者」 (誠実性↑ × 神経症↑)
→ 詳細なステップバイステップ + 失敗しても安全な環境。**チェックリストと「よくある間違い」セクションを必ず含める。**

### 「社交的な実践家」 (外向性↑ × 開放性↓)
→ グループ演習 + 具体的なタスク。**チーム課題で明確なゴールを設定する。**

## プロンプトへの変換例

Big5スコアが `{ O:80, C:30, E:60, A:70, N:40 }` の場合:

```
この学習者は探索的で好奇心旺盛 (O:高) だが計画性は低い (C:低)。
- メタファーや異分野とのつながりを積極的に使用してください
- 長い手順書は避け、体験 → 理論の順序で構成してください
- 短いスプリント(15分以内)で成功体験を早期に提供してください
- 適度な対話性を含めてください (E:中)
- 励ましのメッセージを含めてください (A:高)
```
